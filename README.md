# L1 Cache Info

## Описание

`l1-info` — это небольшая утилита на C++, которая экспериментально определяет основные параметры кэша первого уровня (L1):

- ёмкость (capacity);  
- ассоциативность (associativity);  
- размер строки кэша (line size).

Метод основан на измерении латентности доступа к случайно переставленным участкам памяти.

---

## Сборка

Проект не имеет внешних зависимостей и собирается стандартным `make`.

### Основные цели
| Цель | Описание |
|------|-----------|
| `make` или `make all` | Собрать оптимизированную версию (`-O2`) |
| `make debug` | Собрать отладочную версию с `-O0 -g3 -fsanitize=address` |
| `make clean` | Удалить собранные бинарники |

### Примеры
```bash
# Релизная сборка
make

# Отладочная сборка
make debug

# Очистка
make clean

После сборки доступны исполняемые файлы:

./l1-info — релизная версия

./l1-info-debug — отладочная версия
```

## Запуск

```bash
taskset 1 ./l1-info
```
